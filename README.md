## 🛠 Tech Stack

- HTML, CSS, (Angular, React, Bootstrap)
- Docker
- Jira
- Figma
- GitHub Actions

---

## 📂 Project Structure
- **API endpoints** → `Spotless.Api`
- **UI (MVC/Blazor)** → `Spotless.Web`
- **Entities + services** → `Spotless.Core`
- **EF Core DbContext + migrations** → `Spotless.Infrastructure`
- **Tests** → `Spotless.Tests`
- **Database scripts** → `db/`
- **Documentation** → `docs/`
